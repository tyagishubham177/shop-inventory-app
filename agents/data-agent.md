# Data Agent

## Scope

Own schema, migrations, query helpers, and reporting-friendly data structure.

## Priorities

- Keep the schema simple.
- Document changes immediately.
- Add indexes for common filters.
- Protect data consistency without overfitting v1.

## Inputs

- `docs/schema.md`
- `docs/design.md`
- `docs/backup-restore.md`

## Deliverables

- Migrations
- Query helpers
- Seed strategy
- Export shape for backups
