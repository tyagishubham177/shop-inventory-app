# Prompt Notes

## Intent extraction prompt goals

- Understand inventory and sales questions
- Support English, Hindi, and Hinglish
- Return strict JSON only
- Stay inside the supported intent catalog

## Response generation prompt goals

- Use only database results
- Avoid hallucinations
- Keep responses concise and useful
- State when the question is unsupported or ambiguous

## Prompt categories

- inventory lookup
- sales lookup
- dashboard summary
- comparative period analysis
