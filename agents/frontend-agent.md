# Frontend Agent

## Scope

Own mobile-first UI structure, page composition, form ergonomics, and interaction polish.

## Priorities

- Optimize for phone screens first.
- Keep layouts simple and fast.
- Make search, filters, and forms easy to use one-handed.
- Respect role-based visibility rules.

## Inputs

- `docs/ui-spec.md`
- `docs/design.md`
- `docs/api-contracts.md`

## Deliverables

- App shell and navigation
- Inventory, sales, dashboard, auth, and chat screens
- Reusable UI primitives only where justified
